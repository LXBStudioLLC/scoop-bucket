# LXB Studio Scoop Bucket

[Scoop](https://scoop.sh/) bucket for LXB Studio apps.

## Usage

```powershell
scoop bucket add lxb https://github.com/LXBStudioLLC/scoop-bucket
scoop install lxb/envoy
```

## Apps

| App | Description |
| --- | --- |
| [envoy](https://github.com/LXBStudioLLC/envoy) | Spot ghost jobs before you waste time applying. Local ghost-job scoring with transparent evidence, plus a human-gated apply copilot. |

Both the installer and the portable build are signed by LXB Studio LLC. The Scoop package uses the portable zip; checksums come from the SHA256SUMS.txt published with every release.
